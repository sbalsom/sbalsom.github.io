My Projects

iWell

iWell is an online therapy app, geared towards the European market. Built using Ruby on Rails, Active Record and PostGreSQL, with Pundit and Devise for authentification and authorization, logged in users can search for therapists, book an appointment, and place a video call to their therapist using Twilio. Built in 2 weeks by a team of four at Le Wagon bootcamp in Berlin.

Astrology API

The Astrology API scrapes several publications across the web that have a weekly, daily, or monthly horoscopes section, and compiles the results in a in a public-facing API. The API is accessible to anyone with the url, and allows users to search for and sort horoscopes by publication, date, author name, keywords, sign, and mood.

Is The Moon Void ?

Inspired by isitfridayyet.com, the moon website is a handy tool for astrologers to tell whether the moon is currently void of course. It scrapes two calendars to check for ingresses and current sign the moon is in.

Jamendo Radio
